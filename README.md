# Creative IO

A single page application(SPA) built with react for a creative agency to showcase their portfolio projects.

## Features
- View a list of portfolio projects
- Add new projects dynamically
- Search/filter projects by title
- Responsive design for all screen sizes

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open `http://localhost:5173` in your browser

## Running Tests
Run `npm test` to execute the unit tests

## Technologies used
- React
- Vite
- CSS
- Vitest
- React Testing Library

## Known Limitations
- Projects are not saved after page refresh
